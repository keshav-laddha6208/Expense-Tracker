# Expense Tracker

A simple Python expense tracking app built to practice real-world programming skills.  
The goal is to start with a basic command-line tool and gradually expand it with analytics, budgeting, and possibly a web interface.

---

## What it Does (Current)
- Add expenses
- View all expenses
- Show total spending
- Store data using JSON

---

## Planned Improvements
- Edit / delete expenses
- Monthly summaries
- Budget limits
- Category-wise analytics
- Charts
- SQLite database
- Optional Flask web version

---

## Tech Stack
- Python
- JSON (for now)
- SQLite (planned)
- Flask (maybe later)

---
